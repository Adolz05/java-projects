# Desarrollo Java
Este repositorio contiene mis proyectos enfocados en lógica de programación y desarrollo backend.

## Proyecto Destacado: Gestor de Ficheros Binarios
Aplicación de consola para la gestión de datos persistentes.
- **Funcionalidad**: Genera números aleatorios y los almacena en archivos `.dat`.
- **Técnica**: Uso de `DataOutputStream` y `DataInputStream` para el manejo de flujos binarios.
- **Robustez**: Implementación de manejo de excepciones para fin de fichero (`EOFException`) y errores de I/O.
